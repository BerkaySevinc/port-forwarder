# Port Forwarder

**Port Forwarder** written in C#.</br>
A simple desktop application designed to help configure and manage port forwarding from a router/modem to a local computer.</br>
The application provides an easy-to-use interface for managing network port mappings used in development, testing, or self-hosted services.
</br>
</br>

# Details
- Written in C#.
- Built with **Windows Forms**.
- Provides a graphical interface for configuring **router/modem port forwarding rules**.
- Allows mapping an **external port from the router to a local computer**.
- Useful for development, testing, and hosting local services.
- Designed as a lightweight and simple networking utility.
